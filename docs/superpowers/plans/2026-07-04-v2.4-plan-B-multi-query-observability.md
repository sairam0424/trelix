> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add structured observability to the `MultiQueryExpander` — log whether LLM expansion was used or fell back, how many variants were generated, and persist expansion metadata in the existing `query_telemetry` SQLite table via three new nullable columns.

**Architecture:** Three nullable columns (`expansion_used`, `expansion_variants`, `expansion_elapsed_ms`) are added to `query_telemetry` via an ALTER TABLE migration in `Database.__init__`. `TelemetryWriter.record()` gains an optional `expansion_info` parameter. `retriever.py` captures expansion metrics and passes them to `TelemetryWriter`. `MultiQueryExpander.expand()` gains an `ExpandResult` named return that carries variant count and whether LLM was used.

**Tech Stack:** Python 3.11+, SQLite (via existing `trelix.store.db.Database`), stdlib `dataclasses`.

```python
        self,
```

```python
    def expand(self, query: str) -> ExpandResult:
        """Return ExpandResult with queries=[original] + up to N variants.

        llm_used=False when LLM unavailable or expansion fails.
        elapsed_ms is the wall-clock time of the LLM call (0.0 on fallback).
        Never raises.
        """
        client = self._get_client()
        if client is None:
            return ExpandResult(queries=[query], llm_used=False, elapsed_ms=0.0)
        try:
            from trelix.llm.client import ChatMessage

            system = _MULTI_QUERY_SYSTEM.format(n=self._n)
            t0 = time.monotonic()
            resp = client.complete(
                messages=[ChatMessage(role="user", content=query)],
                max_tokens=200,
                temperature=0.3,
                system=system,
            )
            elapsed_ms = (time.monotonic() - t0) * 1000.0
            variants = [
                line.strip()
                for line in resp.content.strip().splitlines()
                if line.strip()
            ]
            seen: set[str] = {query}
            result_queries = [query]
            for v in variants[: self._n]:
                if v not in seen:
                    seen.add(v)
                    result_queries.append(v)
            return ExpandResult(
                queries=result_queries,
                llm_used=True,
                elapsed_ms=elapsed_ms,
            )
        except Exception as exc:
            logger.debug("Multi-query expansion failed for %r: %s", query, exc)
            return ExpandResult(queries=[query], llm_used=False, elapsed_ms=0.0)
```

Replace the `record` method in `src/trelix/retrieval/telemetry.py`:

```python
    def record(
        self,
        context: RetrievedContext,
        elapsed_ms: float,
        expansion_result: "ExpandResult | None" = None,
    ) -> None:
        """Record a single retrieve() call. No-op when disabled. Never raises."""
        if not self._enabled:
            return
        try:
            from trelix.retrieval.query_expansion import ExpandResult as _ExpandResult

            query = str(getattr(context, "query", ""))
            intent = str(getattr(context, "intent", ""))
            results = getattr(context, "results", [])
            result_count = len(results) if hasattr(results, "__len__") else 0

            exp_used: bool | None = None
            exp_variants: int | None = None
            exp_elapsed: float | None = None
            if expansion_result is not None:
                exp_used = expansion_result.llm_used
                exp_variants = len(expansion_result.queries)
                exp_elapsed = expansion_result.elapsed_ms

            self._db.insert_query_telemetry(
                query=query,
                intent=intent,
                elapsed_ms=elapsed_ms,
                result_count=result_count,
                expansion_used=exp_used,
                expansion_variants=exp_variants,
                expansion_elapsed_ms=exp_elapsed,
            )
        except Exception as exc:
            logger.debug("Telemetry record failed (non-fatal): %s", exc)
```
